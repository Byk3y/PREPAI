# First Screen Assets

This folder contains the splash image for the app's first screen (Duolingo-style).

## Required Image

Place your splash image here with one of these names:
- `splash.png` (recommended)
- `splash.jpg`
- `logo.png`

## Image Guidelines

- **Recommended size**: 300x300px to 512x512px
- **Format**: PNG with transparency or JPG
- **Style**: Simple icon or mascot (like Duolingo's owl)
- The image will be centered on an orange background

## Example

See Duolingo's splash screen for reference - a simple mascot centered on a solid colored background.
